Project setup

I was given 3 different sources of data, SQL, CSV, and JSON for a basic sales database. This also included an ERD for the database.

Implementation

I first had to design a datawarehouse. I decided to use the star schema and created an ERD for my database. I then made an ODS with all the categories I would be using for my database. I was limited to just PANDAS so I needed to use this library in python to parse the data from each source into the correct categories and preprocess any data like dataset into the correct format. After that, I could export my ODS to SQL.
