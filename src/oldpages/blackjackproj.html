Project setup

I was tasked with creating a console blackjack game, with a number of extra features on top.

Implementation

First i create an array of a struct for cards, I also have struct with game settings and a full deck of cards. This also sets up the naming of each card. I then shuffle the deck randomly. The game settings include how many decks used, if the player plays themselves or lets pre defined best practices play for them. They can also choose to see the dealers cards to cheat.

This project took lots of trial and error and making small fixes to make sure the game would work as intended. The extra features were easy to add as I used structs for each game to help, and well commented.
