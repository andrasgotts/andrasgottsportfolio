Project setup

My group was tasked with creating tests for all aspects of a basic banking program and suggesting changes for any issues found.

Implementation

We split the work load between the three people in the group so we all worked on different sections. My focus was on making tests for current accounts. For the tests i created fake current accounts and checked key functions like transferring money worked correctly by making sure the values after the transfers in the accounts were the same as the expected amounts There were some obvious issues in the code, for example when transferring money, there was no check to see if the destination account was open. I did this for all aspects of the current account, found a range of issues in the code and also coded the fixes myself. Overall, the detail of my tests ended up being more than each member of my group, but we still ended up with tests for most aspects of the program.
