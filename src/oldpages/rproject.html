Project setup

Analyse the data from a questionnaire on AI using R

Implementation

I used R to create a large amount of graphs and tables for my analysis report on the data. This ranged from basic bar charts showing which AI tool is most commonly used (ChatGPT), I also used to help draw conclusions on issues like whether the government should be doing more to protect people from AI which involves multiple factors like peoples thoughts on the threat of AI deepfakes, job losses. This data showed that people feared AI and believed the government should be doing more to protect them.

Overall, R proved to be an extremely valuable tool to analyse data, and I was able to analyse so much about the questionnaire thanks to it.
