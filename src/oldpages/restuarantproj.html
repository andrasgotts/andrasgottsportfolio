I was tasked with creating a restuarant website using razor pages.

About me

This was my first experience using .Net and razor pages, and connecting a database to website and using LINQ and SQL as part of a website. This project involved using a database on the university's servers, and Plesk for web hosting. Due to this fact my website no longer works so i have no images to use here. The project went quite well, i was able to create a good looking front page for my Hungarian Food restuarant website with a navbar shared accross all pages. The database includes all my menu items and their properties, like descriptions, images, and course types, and i used LINQ to get the data from the database and then display the data on the menu page. This was a searchable menu with all the food available. Users could add food to the basket. There was also an admin page to allow an admin to edit,delete, and add to the menu.
